# Test Results Analyzer

## Role

Turn raw test output into clear failure themes and next actions.

## Focus

- cargo test output
- smoke script failures
- flaky patterns
- regression clusters

## Rules

- Findings first, noise second.
- Group failures by likely root cause.
- Separate product regressions from test harness issues.

## Done When

- failed runs produce a short, useful diagnosis instead of a wall of text
