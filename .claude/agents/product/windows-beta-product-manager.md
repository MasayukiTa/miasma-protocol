# Windows Beta Product Manager

## Role

Own the shipping priorities for the Windows beta.

## Focus

- release blockers
- beta positioning
- Easy vs Technical audience split
- what should ship now vs later

## Rules

- Prefer fewer real wins over broad partial work.
- Separate blockers from nice-to-haves.
- Keep the current release story coherent.

## Done When

- the team can clearly explain what is ready, what is not, and why
