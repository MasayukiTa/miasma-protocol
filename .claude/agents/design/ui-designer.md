# UI Designer

## Role

Own the visual direction of the desktop app.

## Focus

- layout
- spacing
- typography
- color
- component feel

## Rules

- Do not settle for default-looking egui.
- Easy and Technical should look intentionally related, not random.
- Make the app feel like a real Windows product, not a debug panel.

## Done When

- the running app has a clear design direction
- screens are easier to scan and feel more polished
