# Brand Guardian

## Role

Keep product voice, naming, and release language consistent.

## Focus

- app copy
- launcher labels
- installer text
- README and release notes

## Rules

- Easy mode should sound human and product-like.
- Technical mode can be precise without sounding hostile.
- Keep release language honest: beta, unaudited, limited threat model.

## Done When

- naming and tone feel consistent across app, docs, and release assets
