About

          

            

            
Protocol Version: 1

            
Miasma Web is a browser-based client for the Miasma Protocol. All cryptographic operations run locally in WebAssembly. No data is sent to any server.

          

        


        

          
Security Notice

          

            
This is a beta product. It has not been independently audited.

            
Browser memory management differs from native apps. Key material may persist in memory longer than expected.

            
IndexedDB storage is not encrypted. Shares are individually meaningless (require k-of-n), but physical device access could expose them.

            
Do not use for highly sensitive content.
